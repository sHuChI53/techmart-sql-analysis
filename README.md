# TechMart SQL Analysis

## Project Overview

This project analyzes TechMart retail business data using SQL. The objective is to extract meaningful business insights by writing optimized SQL queries on structured relational datasets. The project focuses on data exploration, aggregation, filtering, joins, and performance-oriented query writing.

## Objective

The goal of this assignment is to solve business problems using SQL and demonstrate proficiency in querying relational databases. The analysis covers customer behavior, product performance, sales trends, and operational insights.

## Tools Used

* MySQL Workbench
* SQL
* Relational Database Concepts

## Dataset Description

The TechMart database consists of multiple related tables representing:

* Customers
* Orders
* Products
* Sales Transactions
* Categories
* Regions
* Revenue Data

These tables are connected using primary and foreign key relationships to enable relational analysis.

## Tasks Performed

* Wrote SQL queries to answer 25 business questions
* Used filtering with WHERE clause
* Applied aggregation functions (SUM, AVG, COUNT, MAX, MIN)
* Performed INNER JOIN, LEFT JOIN, and multi-table joins
* Used GROUP BY and HAVING clauses
* Implemented ORDER BY for sorting results
* Applied subqueries for advanced analysis
* Used aliases for readability
* Structured queries with proper comments
